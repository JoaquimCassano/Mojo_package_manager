# Project Mojo_package_manager

A package manager (like Pip, Npm and Bun) for Mojo Lang!

## Requirements
- [Mojo](https://developer.modular.com/download) >= 0.2.1
- [Python](https://www.python.org/downloads) >= 3.10
- [Git](https://git-scm.com/downloads) >= 2.34.1


## Installation

To install the project, follow these steps:


- Run the command below:
```
python3 -m pip install mpak
```

## Usage

To use the project, you can run the following commands:

- `mpak install <package_name>`: Downloads and installs a package.
- `mpak uninstall <package_name>`: Uninstalls a package from your project.
- `mpak  upload`: Uploads your package to mpak, allowing everyone to download and install it.
- `mpak update <package_name>`: Updates your package in mpak.



*Make sure to replace `<package_name>` with the actual name of the package you want to install or uninstall.*
